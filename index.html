📱 iOS Setup Guide

🎯 Quick Fix: Files App Method

1. Open Files app on your iPhone/iPad
2. Find the downloaded HTML file
3. Tap and hold the file
4. Tap "Share" → "Copy to Safari"

📧 Alternative: Email Method

1. Email yourself the HTML file
2. Open email on your device
3. Tap attachment → "Open in Safari"

🌐 Best Solution: Use Web Version

Upload to free hosting:
• GitHub Pages (free, easy)
• Netlify Drop (drag & drop)
• Google Drive (share as web page)

⚙️ iOS Settings Check

1. Go to Settings → Safari
2. Ensure "JavaScript" is enabled
3. Check "Block Pop-ups" is off
